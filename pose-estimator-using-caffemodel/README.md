
### Usage Examples :
Put the test file (iamge or video) under the same directory   
   
 - `python3 estimator_multi_people.py --image=test.jpg`   
 - `python3 estimator_multi_people.py --video=test.mp4`   
 - if no argument provided, it starts the webcam.
